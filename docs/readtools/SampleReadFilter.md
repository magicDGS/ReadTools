## Description

Keep only reads for a given sample.
 Matching is done by case-sensitive exact match.

## Arguments

### Required Arguments

| Argument name(s) | Type | Description |
| :--------------- | :--: | :------ |
| `--sample`<br/>`-sample` | Set[String] | The name of the sample(s) to keep, filtering out all others |
