## Description

Filters out reads with more than a threshold number of N's

## Arguments

### Optional Arguments

| Argument name(s) | Type | Default value(s) | Description |
| :--------------- | :--: | :--------------: | :------ |
| `--ambigFilterFrac` | double | 0.05 | Threshold fraction of ambiguous bases |
