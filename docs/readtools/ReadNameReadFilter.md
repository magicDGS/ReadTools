## Description

Keep only reads with this read name.
 Matching is done by case-sensitive exact match.

## Arguments

### Required Arguments

| Argument name(s) | Type | Description |
| :--------------- | :--: | :------ |
| `--readName`<br/>`-readName` | String | Keep only reads with this read name |
