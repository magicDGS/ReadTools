## Description

Keep only reads with this read name.

 <p>Matching is done by case-sensitive exact match.</p>

## Arguments

### Required Arguments

| Argument name(s) | Type | Description |
| :--------------- | :--: | :------ |
| `--readName` | String | Keep only reads with this read name |
