## Description

Keep only reads whose strand is forward or reverse

## Arguments

### Required Arguments

| Argument name(s) | Type | Description |
| :--------------- | :--: | :------ |
| `--keepReverse`<br/>`-keepReverse` | Boolean | Keep only reads on the reverse strand |
