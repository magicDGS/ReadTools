## Description

Only use reads from the specified read group.
 Matching is done by case-sensitive exact match.

## Arguments

### Required Arguments

| Argument name(s) | Type | Description |
| :--------------- | :--: | :------ |
| `--keepReadGroup`<br/>`-keepReadGroup` | String | The name of the read group to keep |
